Project Description.

Project Skylar is for educational use only and is not intended for commercial distribution. The project was proof-of-concept on designing and implementing distributed systems infrastructure for the backend servers. The idea was to allow its online servers to clone themselves and deploy new ones as soon as the game servers couldn't handle more traffic and additional players requests.

Now after finishing with all the backend nonsense and my studies, I decided to release the source code KABoOm!! :D

Enjoy coding your way through it. I made sure to document every script before leaving it behind. Consider adding more features and dont forget sharing it with everyone!
